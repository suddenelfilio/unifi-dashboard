## Features Summary (bullet points)

✅ Real-time monitoring with 15-second auto-refresh
✅ Smart notifications for client connections/disconnections
✅ Site availability inference based on device status
✅ Collapsible site details and filter panel
✅ Sound alerts with mute toggle
✅ Responsive design for all devices
✅ Docker deployment ready
✅ Environment-based configuration
✅ Secure BFF proxy for CORS handling
✅ Non-root container execution
✅ Support for unlimited UniFi sites
✅ Network health metrics and visualization

## Quick Start (one-liner)
```bash
git clone <repo-url> && cd UI-Monitor && cp .env.example .env && npm install && npm start
```

## Docker Quick Start (one-liner)
```bash
docker run -d -p 3000:3000 -e UNIFI_API_KEY=<your-key> -e UNIFI_API_URL=<your-url> <image-name>
```
